## 1. 背景課題
複数のプレス機械を用いた金属部品加工の高サイクル生産を行っているS社では、
摩耗や潤滑不良など微細な設備異常がコスト損失を生じる課題を抱えている。
加工品質の安定化と設備保全の高度化を実現するため設備故障要因である異常
(特に潤滑不足)を早期に検知することを目指している。<br>

## 2. 評価指標及び課題
**評価指標**：
異常検知用の評価指標として以下を選択(任意)<br>
- **ROC-AUC**：新陽性率-偽陽性率曲線の下の面積
- **PR-AUC**：Precision-Recall曲線の下の面積
- **F1スコア**：調和平均

|No. |AI分類モデル |ROC-AUC	|PR-AUC	|F1スコア|
|:-----|:-----|:-----|:-----|:-----|
|(1)	|LightGBM	|0.7239 	|0.4724 	|0.4256|
|(2)	|LightGBM	|0.9533 	|0.9158 	|0.9328|
|(3)	|LightGBM	|0.9652 	|0.9237 	|0.9354|
|(4)	|KNN	|0.9460 	|0.8740 	|0.8400|
|(5)	|PCA	|0.8910 	|0.7280 	|0.6730|
|(6)	|iForest	|0.9220 	|0.7900 	|0.7290| 

(1)は前処理無しの素のデータで実行。(2)はzスコア補正, IQR補正, 移動平均, 
スライド窓(SW=30)統計量で実行。(3)は(2)に移動平均を追加して実行。
(4),(5),(6)は(3)の加工データに対して各モデルを適用して実行。<br>
